HTML "block-level" elements

Tag Name Description
<address> The Contact Address element Provides contact information for a person or people, or for an organization.
<article> The Article Contents element A forum post, a magazine or newspaper article, or a blog entry, a product card, a user-submitted comment, an interactive widget or gadget, or any other independent item of content.
<aside> The Aside element Asides are frequently presented as sidebars or call-out boxes. Do not use it to tag parenthesized text, as this kind of text is considered part of the main flow.
<div> The Content Division Element The generic container for flow content. It's used to group content so it can be easily styled.
<nav> The Navigation Section Element It represents a section of a page whose purpose is to provide navigation links, either within the current document or to other documents.
<section> The Generic Section element It is a generic sectioning element, and should only be used if there isn't a more specific element to represent it. Each <section> should be identified, typically by including a heading (<h1>-<h6> element) as a child of the <section> element, wherever possible.
